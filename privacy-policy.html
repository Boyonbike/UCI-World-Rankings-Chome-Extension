Privacy Policy – UCI World Rankings Tool

Last updated: February 2026

UCI World Rankings Tool is a Chrome browser extension that helps users view and filter UCI cycling rankings. This privacy policy explains how the extension handles data.

Data Collection

This extension does not collect, store, transmit, or share any personal data. No information about you, your browser, or your browsing activity is ever sent to any server or third party.

What the Extension Does

The extension reads and modifies the content of pages on dataride.uci.ch solely to filter and display ranking information within your browser. This processing happens entirely on your own device and is not recorded or transmitted anywhere.

Permissions

The extension requests the following permissions:

  • activeTab / tabs: To detect when you are on a supported UCI rankings page and to inject filtering functionality.
  • scripting: To run scripts that filter and extend the rankings table on the page.
  • Host permission (dataride.uci.ch): To limit the extension's access strictly to the UCI rankings website. It does not access any other websites.

No permission is used to collect or transmit data.

Third Parties

This extension does not use any analytics, advertising, or third-party services of any kind.

Changes to This Policy

If this policy changes in the future, the updated version will be posted at this URL with a revised date.

Contact

If you have any questions about this privacy policy, please open an issue on the extension's GitHub repository or contact the developer directly via the Chrome Web Store listing.
